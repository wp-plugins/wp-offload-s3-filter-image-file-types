== Description ==

Filters the plugin [WP Offload S3](https://wordpress.org/plugins/amazon-s3-and-cloudfront/) by stopping images only from uploading to the S3 bucket.

== Installation ==
Installing "WP Offload S3 Filter Image File Types" can be done either by searching for "WP Offload S3 Filter Image File Types" via the "Plugins > Add New" screen in your WordPress dashboard, or by using the following steps:

1. Download the plugin via WordPress.org or
2. Upload the ZIP file through the 'Plugins > Add New > Upload' screen in your WordPress dashboard.
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. Done

== Frequently Asked Questions ==

= Q.1 What image file formats does it stop from uploading to S3? =

A.1 The following formats have been filtered:
 * .bmp
 * .gif
 * .jpg
 * .jpeg
 * .jpe
 * .png
 * .tiff
 * .tif
